## java.lang.Math 클래스

- 수학에서 자주 사용하는 상수 및 함수들을 미리 구현해놓은 클래스

- Math 클래스의 모든 메소드는 클래스 메소드(static method)이므로, 객체를 생성하지 않고 바로 사용할 수 있음

- java.lang 패키지에 포함되어 제공(별도 import 하지 않아도 됨)

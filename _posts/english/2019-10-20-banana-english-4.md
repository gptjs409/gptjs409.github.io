## 수업 내용 정리

#### 지난 번 숙제시

- 5형식을 배웠기 때문에 대충 틀을 잡을 수 있었음

- 하지만 다음 부분에 대해서는 고민하게 됨

  - 각 자리에 무슨 단어를 쓸 지
  
  - 어떤 모양으로 쓸 지
  
  - 뜻을 알지만 뭘 넣어야 할지
  
- 위 부분을 고민할 때 알아야 하는 문법에 대한 학습

<br>

#### 5형식의 문장 구성요소 4가지

- 지난 번 강의에서 배웠던 부분

  - 1형식 S+V \[완전자동사]
  
  - 2형식 S+V+SC \[불완전자동사]
  <br>= S+V+C
  
  - 3형식 S+V+O \[완전타동사]
  
  - 4형식 S+V+I.O+D.O \[수여동사]
  <br>= S+V+O+O
  
  - 5형식 S+V+O+OC \[불완전타동사]
  <br>= S+V+O+C
  
- 필요한 4가지

  - S 주어, V 동사, O 목적어, C 보어
  
- V \[서술어(술어)]

  - 품사 중 '동사'가 들어가는 자리
  
  - '동'動 : 움직일 동
  
  - 동작 / 상태 / 행위 등을 나타냄
  
  - 종류 : 동작 동사 / 상태 동사
  
  - 동작동사 Action verb : 동작으로 표현 가능한 동사
  <br>eat(먹다) / run(달리다) / sing(노래하다) / walk(걷다)
  
  - 상태동사 State verb : 동작으로 표현할 수 없는 동사
  <br>love(사랑하다) / hate(싫어하다) / want(원하다) / have(가지다)
  
- S \[주어]

  - 품사 중 '명사'가 들어가는 자리
  
  - 동사, 행위의 주체가 옴
  
- O \[목적어]

  - 품사 중 '명사'가 들어가는 자리
  
  - 동사, 행위의 대상이 옴
  
  - 주로 \~를 / \~을 이라고 해석

- C \[보어]

  - 품사 중 '명사'나 '형용사'가 들어감
  
  - S의 상태(형용사) 또는 동격(명사)이 옴
  
  - 2형식 보어는 주로 '\~이다/\~되다'라는 동사와 함께 씀
  
<br>

#### 8품사

- 8품사

  - 단어를 문법적 기능에 따라서 분류해 놓은 것
  
  - 종류 : 명사 / 대명사 / 동사 / 형용사 / 부사 / 전치사 / 접속사 / 감탄사
  
  - 기본적인 역할 : 문장에서 S/V/C/O 자리에 들어감
  
- 주요 8품사

  - 명사 / 대명사 / 동사 / 형용사
  
<br>

#### 주요 8품사
  
- 명사

  - 명사名詞; 이름 '명'
  
  - 이름을 가진 모든 것들
  
  - ex) 아티 / 펜 / 사랑 / 물 / 싸움 ... 
  
  - S / C / O / 전명구 자리에 씀
  
- 대명사

  - 대명사大名詞; 대신할 '대'
  
  - 명사를 대신해서 씀
  
  - 예시1) I have one American friend.
  <br>~~My American friend~~ She \~
  <br>~~My American freind~~ She \~
  
  - 예시2) I bought a computer.
  <br>~~My computer~~It was cheap.
  <br>~~My computer~~I was expensive.
  
  - 영어는 말의 반복이나 단어의 중복된 사용을 꺼리고 싫어함
  <br>말이 반복적으로 쓰여서 비효율적인 사용을 막음
  
  - ex) 아티=He, 바나나=She, 물건=It, 우리=We, 사람들/물건들=They
  
  - S / C / O / 전명구 자리에 씀
  
  - 주의사항) 자리에 따라 모양이 바뀜
  <br>S자리에 쓸 때는 He / She / We / They
  <br>O자리에 쓸 때는 him / her / us / them
  
- 동사

  - 동사動詞; 움직일 '동'
  
  - 동작이나 상태를 나타냄
  
  - 모양은 크게 be동사(상태 동사) / 일반동사(동작동사, 상태동사)로 나뉨
  
  - be동사(상태동사)
  <br>am/are/is/was/were/been
  
  - 일반동사-동작동사 : 동작으로 표현 가능한 동사
  <br>eat/run/sing/walk
  
  - 일반동사-상태동사 : 동작으로 표현 불가능한 동사
  <br>love/hate/want/have
  
  - V 자리에 씀
  
- 형용사

  - 형용사形容詞; 모양 형, 얼굴 용
  
  - 명사의 성질이나 상태를 나타냄
  
  - 명사를 꾸며주고 수식함
  
  - 주로 'ㄴ' '\~한'으로 끝남
  
  - ex) 예쁜 / 귀여운 / 흥미로운 / 짜증난
  <br>pretty / cute / interesting / annoyed
  
  - 명사를 수식하는 방식에는 두 가지가 있음
  <br>1) 명사의 앞/뒤에서 명사 꾸미기(전치수식/후치수식) : 한정적 용법 → 한 명사가 가지고 있는 범위를 한정시킴
  <br>2) 보어자리(주격보어/목적격보어)에 위치하기 : 서술적 용법 → 보어자리에(특히 주격보어) 오는 것을 서술적 용법이라함
  <br>주어(또는 목적어)의 상태를 서술
  
  - 전치수식 : 명사의 앞에서 명사를 꾸미는 방법
  <br>**a smart** girl / **an expensive** car
  
  - 후치수식 : 명사의 뒤에서 명사를 꾸미는 방법 (주로 -thing / -body / -one / -where)
  <br>something **spectial** / something **cheap**
  <br>anything **new** / somebody **else** / somewhere **hot**

  - 주격보어(SC) - 2형식
  <br>She is **gorgeous**. / It is **useful**.
  
  - 목적격보어(OC) - 5형식
  <br>I consider him **smart**.
  
<br>

#### 형용사 추가

- '관사' a / an / the

  - 명사를 꾸며주기 때문에 형용사임

- '소유격' her / his / my / their / our

  - 명사를 꾸며주기 때문에 형용사임
  
- 결론 명사를 꾸며주면 다 형용사임

- a cute dog : 형용사 두 개

- 형용사는 여러 개로 명사를 꾸밀 수 있음(대신 순서가 존재)

  - 순서가 너무너무 중요한 건 아니지만, 원어민들은 이 순서를 자연스럽게 사용하기 때문에 바꿔서 쓰면 어색하게 들림

<br>

#### 형용사 순서(맞지 않아도 되지만 어색해짐)

- 우선순위 1) 전치 한정사

  - all / both / half / double(배수사)

- 우선순위 2) 한정사

  - a / the / these / his / her / some / any / no

- 우선순위 3) 서수

  - first / second / third

- 우선순위 4) 기수

  - one / two / three

- 우선순위 5) 성질형용사

  - (크기-성질-신/구-색-기원-재료)
  
- 형용사를 여러 개 쓸 때는 ',(콤마)'로 이어줌
  
- 예시

  - A large, beautiful, old house
  <br>관사, 크기, 성질, 신/구, 명사
  
  - A large, beautiful, old, brown table
  <br>관사, 크기, 성질, 신/구, 색, 명사
  
  - The mysterious, American woman
  <br>관사, 성질, 기원, 명사

## 숙제

- 오늘 배운 각 품사에 대해서 정리하고 필기하기

  - 블로그 포스팅
  
- 오늘 배운 품사 정의내려 한 줄 요약

  - 명사 : 이름을 가진 모든 개체
  
  - 대명사 : 명사를 대신해 대중적으로 표현하는 것
  
  - 동사 : 개체의 상태나 행위
  
  - 형용사 : 개체를 꾸며주는 말
  
- 각 품사(명사/대명사/동사/형용사) 종류를 5개 이상 찾아보기

  - 명사 : Sun, Moon, Star, Cloud, Sky
  
  - 대명사 : I, We, You, She, He, They, It
  
  - 동사 : Move, Find, Stop, Save, Keep
  
  - 형용사 : Good, Easy, Old, White, Young
